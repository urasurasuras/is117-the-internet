IP Addresses and DNS

IP addresses are essentially addresses that makes sense in the architecture of the Internet.
IPv4 is a type of address that is 32 bits divided into 4 octets, representing a hierarchical structure.
Although IPv4 has the capability to form more than 4 billion unique IP addresses, that wasn't enough as time progressed.

127.0.0.1

This is an example of IPv4, being the loopback address that represents the local machine.

Hence we are currently in a transition to a more robust system called IPv6, which is formed of 128 bits, that looks more like this:

2001:0db8:0000:0000:0000:ff00:0042:8329

A Domain Name Server translates domain names into IP Addresses for the clients computers to access.
